# Root Exit · 界域行者

## Introduction · 介绍
Root Exit 代表Linux系统中，从危险的超级用户世界退出，放弃超级权限，遵守最小权限原则，，进入更安全、更有秩序的系统

“Root Exit” is more than leaving root access behind. It represents a conscious choice to give up absolute privilege, follow the Principle of Least Privilege, and build systems that are safer, more resilient, and governed by order.


## Manifesto · 宗旨
“权限应该有边界，有领域”

Power requires boundaries. Privileges belong within domains.

## About · 组织说明
界域行者就是这样一群遵守了这个宗旨的人聚合在一起的结果，所编写的软件都应当遵循这个原则

Boundary Walkers are the result of like-minded individuals united by this philosophy. Every piece of software we create should embody and follow these principles.
